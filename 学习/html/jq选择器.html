层次选择器

$("div > span") 选取元素下元素名是 span 的子元素

$(".one+div")
$(".one").next("div") 选取class为one的下一个 div 同辈元素

$("#two~div")
$("#two").nextAll("div") --------后面的所有 div 同辈元素

不同点：$("#prev~div")只能选择ID元素后面的同辈 div元素。而siblings()方法与前后位置无关，只要是同辈节点就都能匹配。

基本过滤选择器

$("input:not(.myClass)")选取class不是myClass的 input 元素

$("input:even") 选取索引是偶数的 input 元素
$("input:odd")----------奇数的-----------

$("input:gt(1)")选取索引大于1的 input元素（大于1，而不包括1）
$("input:lt(1)")------小于-----------------------------

$(":focus")选到当前获取焦点的元素

内容过滤选择器

$("div:contains('我')") 选取含有文本"我"的div元素

$("div:has(p)")选取含有 p 元素的 div 元素


属性过滤选择器

$("div[id]")选择拥有属性 id 的元素

$("div[title=test]")选取属性title为"test"的div元素
$("div[title!=test]")-----------不等于-----------
$("div[title*=test]")-----------含有-----------------
$("div[title^=test]")选取属性 title 以"test"开始的 div元素
$("div[title$=test]")---------------------结束的-------

$("div[title|="en"]")选取属性 title 等于en 或以 en为前缀（该字符串后跟一个连接字符'-'）的元素

$("div[title`="uk"]")选取属性 title 用空格分隔的值中包含字符 uk 的元素

$("div[id][title$='test']")选取拥有属性 id ，并且属性 title 以"test" 结束的 div 元素

子元素过滤选择器

:nth-child() :nth-child(even) :nth-child(3n+1)
不同点：:eq(index)是从 0 算起，而:nth-child的index是从1开始的。

:only-child 如果某个元素是它父元素中唯一的子元素，那么将会被匹配。如果父元素中含有其他元素，则不会被匹配。

表单对象属性过滤选择器

$("#forml:enabled")选取 id 为"forml"的表单内的所有可用元素
$("#form2:disabled")--------------------------不可用元素

$("input:checked")选取所有被选中的 input 元素（单选框、复选框）

$("select option:selected")选取所胡被选中的选项元素
